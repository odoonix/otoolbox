# Folder structure

- 'odoo/' folder contains the Odoo source code, which is the main application.
  (This is read only and should not be modified.)
- 'oca/' folder contains the Odoo Community Association (OCA) modules, which are
  additional modules developed by the community. (This is read only and should not
  be modified.)
- 'odoonix/' folder contains the Odoonix modules, which are custom modules developed
  by Odoonix. (This is the main focus for development and can be modified.)
- 'moonsunsoft/' folder contains the Moonsunsoft modules, which are custom modules
  developed by Moonsunsoft. This is a clone of the 'odoonix/' folder and is used for
  testing and development purposes. (This is read only and should not be modified.)
